## Set your APIkey and Domain

Use [cocoapods-keys](https://github.com/orta/cocoapods-keys)

```
$ gem install cocoapods-keys
$ pod keys set "SkyWayApiKey" "<your APIkey>"
$ pod keys set "SkyWayDomain" "<your Domain>"
$ pod install
```
